.. title: XSLT
.. type: text

XSLT is an abberviation for Extensible Stylesheet Language Transformations.

XSLT (XSL Transformations) is a technology which transforms XML documents into
HTML, JSON, PDF, Plain Text, XHTML, and (modified) XML documents.

It works over both, servers and clients.

The documents of this publictaion are formed as XML files (Atom Syndication
Format or OPML Collection), and conveyed as XHTML documents; these XHTML
documents of this publication are produced by XSLT stylesheets.
