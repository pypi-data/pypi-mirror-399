# Semantic Search Resume Watcher

Resume the file watcher to restart automatic reindexing of changed files.

## Action

1. Call `mcp__semantic-search__resume_watcher` tool
2. Confirm to the user that the watcher is running again
3. Inform them that file changes will now be automatically indexed
