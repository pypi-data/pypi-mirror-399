This module adds **Titles** on partners (Madam, Mister, etc.)

The *Title* field on partners was native up to Odoo 18.0 and was dropped
in 19.0. The goal of this OCA module is to restore the feature.

   :local:

Configuration
=============

Go to the menu *Settings > Technical > Contact Titles > Contact Titles*
to add, update, archive or delete partner titles.

Titles are for physical persons only, not for companies.
