*NiQuery*
=========

**Neuroimaging dataset query, analysis, and aggregation tool for
research hypothesis testing**.

*NiQuery* is a principled tool to query open neuroimaging datasets,
extract relevant information, and create new DataLad datasets for
research hypothesis testing.
