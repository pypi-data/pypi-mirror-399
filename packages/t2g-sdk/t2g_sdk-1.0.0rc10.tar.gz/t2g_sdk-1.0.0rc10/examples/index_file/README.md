# Index File Example

This example demonstrates how to use the `T2GClient` to index a file with an associated ontology and save the output to a Neo4j database.

## How to Run

1.  **Set up your environment:**
    Make sure you have the required environment variables set for the `T2GClient`. See the main [README.md](../../README.md#configuration) for more details.

2.  **Run the script:**
    ```bash
    python index_file.py
    ```
