
This module adds the “Qualification log” tab to the contact form. The user indicates the value of three parameters:
   - Value for money.
   - Service.
   - Attention given.

When saved, the system calculates the average of the Qualification in the “partner quality” field and adds a record to the history with the following data:
   - Qualification date.
   - User who made the Qualification.
   - Value for money.
   - Service.
   - Attention given.
   - Partner quality.
